## ✨ About The Project

**Anmol's Apothecary** is a production-grade, India-focused user registration and authentication system — built with pure HTML, CSS, and JavaScript. No frameworks. No backend. Just clean, thoughtful frontend engineering.

Designed to feel like a **real wellness brand portal**, every detail has been crafted with care — from the olive-green palette to the real-time password validation, from the auto-age calculator to the credential-saving flow that mirrors Google Password Manager.

> *"Not a demo. Not a template. A complete product experience."*

## 🗺️ Features At A Glance

### 📋 Registration Form
- ✅ **First / Middle / Last Name** — First & Last mandatory
- ✅ **Father's Name** field
- ✅ **Date of Birth → Auto Age Calculation** — accurate to the day
- ✅ **Phone Number** with international country prefix selector
- ✅ **Email Validation** — must be a valid `.com` address
- ✅ **Address Lane, Pincode, Country** (India 🇮🇳 locked)

### 🗾 India-Only Location System
- ✅ **All 28 Indian States** in the dropdown
- ✅ **All 8 Union Territories** included
- ✅ **Dynamic City Dropdown** — cities update instantly on state selection
- ✅ **No partial lists** — every state has multiple major cities

### 🔐 Password Security
- ✅ **Real-time password rules** — turn green as you type
- ✅ Minimum 8 characters
- ✅ At least 1 uppercase letter
- ✅ At least 1 number
- ✅ At least 1 special character (`! @ # $ % ^ & *`)
- ✅ **Confirm password** live match indicator
- ✅ **Show / Hide** password toggle on both fields

### 🔑 Smart Credential System
- ✅ **Save Credentials Popup** appears right after account creation
- ✅ Works like **Google Password Manager** — tap to auto-fill on login
- ✅ Credentials stored in `localStorage` for seamless return visits
- ✅ Auto-prefill on login page after registration

### 🔓 Login Page
- ✅ **Saved Credentials Banner** — one tap fills email + password
- ✅ **Remember Me** checkbox
- ✅ **Forgot Password** modal with email lookup
- ✅ **CAPTCHA** security check activates after 3 failed attempts
- ✅ **3-second success popup** on successful login

### 🎨 UI & Design
- ✅ Olive-green brand theme throughout
- ✅ Cormorant Garamond + DM Sans typography
- ✅ Fully responsive — fits all screen sizes
- ✅ Smooth animations and micro-interactions
- ✅ Business-grade professional layout
