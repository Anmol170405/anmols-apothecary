## What Is This?

A clean, India-focused **registration and login system** — built from scratch with zero frameworks. Designed to feel like a real product, not a student project. Every field, every validation, every interaction was considered deliberately.

No React. No Node. No fluff. Just HTML, CSS, and JavaScript doing exactly what they're supposed to do.

## Core Features

**Registration**
- First, Middle, Last name — first & last required
- Father's name, DOB with auto age calculation
- Phone with country prefix · Email must end in `.com`
- Address, Pincode, India-locked country field

**Location System — India Only**
- All 28 states + 8 Union Territories
- City dropdown updates dynamically on state selection
- Every state has multiple major cities — no partial lists

**Password**
- Rules display on focus — turn green as each condition is met
- Minimum 8 chars · 1 uppercase · 1 number · 1 special character
- Confirm password live match indicator
- Show / Hide toggle on both fields

**Credential Flow**
- Save credentials prompt appears immediately after account creation
- Saved credentials banner on login page — one tap to auto-fill
- Mirrors the feel of Google Password Manager
- Remember Me + Forgot Password modal included

**Security**
- CAPTCHA activates after 3 failed login attempts
- localStorage used for demo-level credential storage
